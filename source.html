How this website is built

The website separates curated public pages from generated source coverage. This makes the public narrative readable while keeping the PDF traceable and volatile public links explicit.

Curated pages

Edited public pages live in src/content/pages/*.json.

Shared site data

Source atlas

scripts/extract_pdf.py extracts all PDF pages into src/content/pdf-pages.json.

Static generator

src/build.mjs renders the root HTML files and atlas pages for GitHub Pages.

No runtime framework

The site is plain HTML, CSS, and JavaScript for durability and simple hosting.
