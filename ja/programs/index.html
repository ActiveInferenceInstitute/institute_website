インスティテュートプログラムは、ボランティア、インターンシップ、メンターシップ、フェローシップ、パートナーシップ、慈善活動、助成金、オープンソースの道筋を通じて興味を参加へと変えます。すべてのプログラムは、背景や知識レベルに関係なく誰にでも開かれています。

ボランティア

ボランティア活動は参加のための最も包括的な入り口です。ボランティアは、利用可能な時間、背景、スキルに応じて、学習グループ、プロジェクト、運営、執筆、分析、メディア制作、コミュニティサポートなど様々な分野に貢献します。アクティブインファレンスの事前知識は必要ありません—多くのボランティアは、会議への参加や録画の視聴から始め、現在の能力に合わせて貢献しています。

Discord inviteCommunityVolunteer page

インターンシップ

インターンシップは、特定のプロジェクトや機能に深く貢献したい参加者のための構造化された関与を提供します。インターンたちは、文書化、分析、コード、教育資料、または運営サポートなどの定義された成果物に取り組み、プロジェクトの実践とコミュニティへの関与を通じてスキルを習得します。事前の経験に関係なく学習がプロセスに組み込まれています。

InternshipParticipateInternship page

メンターシップ

メンターシップは、アクティブインファレンスやインスティテュートに精通している経験豊富な参加者と、まだ初期段階の人々を結びつけるものです。メンターは、指導、フィードバック、共同プロジェクト、プロフェッショナルなつながりを通じて、メンティーの成長をサポートします。メンターシップは形式的なもの（明確な目標と定期的なチェックインがあるもの）や、コミュニティ参加から自然に生じる非公式なものがあります。

Mentorship pathwayParticipateMentorship page

フェローシップ

フェローシップ・ウェイは、定期的な進捗報告を提供し、研究所やエコシステムに関連する研究、アプリケーション、またはパブリックリソースを開発する貢献者を支援します。フェローは、特定のプロジェクトや作業領域に対する継続的なコミットメントを行い、コミュニティに対して定期的に更新を共有し、論文、ソフトウェア、教育資料、またはプロジェクト成果物などのパブリックアウトプットを生成します。

Fellows pathwayParticipateFellowship page

助成金

研究所は、アクティブインファレンスの研究、教育、応用に関連する助成金提案や資金調達活動を支援しています。参加者やプロジェクトは、助成金準備、機関所属、共同資金提案において研究所の支援を求めることができます。研究所は資金機会を把握し、定期的なコミュニケーションを通じて共有します。

Weekly updatesCommunityGrants page

パートナーシップ

パートナーシップは、組織、研究グループ、資金提供者、協力者を具体的な共通の取り組みに結びつけます。パートナー組織は、イベントを共同開催したり、共同で論文著述を行ったり、リソースを共有したり、インフラを提供したり、研究および教育プロジェクトで協力したりすることができます。パートナーシップとは、時間、注意力、専門知識、資金、機関サポートの実践的な交換です。

PartnershipParticipatePartnership page

慈善活動

非営利団体として、Active Inference Instituteは個人および組織から税控除可能な寄付を受け付けています。慈善的な支援は、公共教育、研究インフラ、イベント、コミュニティツール、エコシステムの管理を資金面で支えています。

Donation pageSupportPhilanthropy page

オープンソース

オープンソースへの貢献は、エコシステムのサポート機能です：再利用可能な作業を公開し、プロジェクトのインフラストラクチャを改善し、共有技術リソースを発見可能かつ保守可能にします。研究所は、アクティブインファレンスの実装、ツール、データセット、オントロジー、教育資料のためのオープンソースリポジトリをホストし、貢献しています。
