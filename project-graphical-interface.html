The Graphical Interface project develops visual and interactive tools for working with Active Inference models. It focuses on making model structure, dynamics, and outputs more accessible through well-designed graphical interfaces and visualization layers.

Overview

The Graphical Interface project creates user-facing tools for exploring, building, and visualizing Active Inference models. Good visual interfaces lower the barrier to entry for working with generative models and make model structure more interpretable to practitioners and researchers.
