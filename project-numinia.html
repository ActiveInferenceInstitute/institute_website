Numinia is an Ecosystem project developing an autonomous AI and educational adventure game grounded in Active Inference principles. The first mission embeds Active Inference in the values of the Numinia agent, creating an environment where players and agents co-learn through play.

Overview

Numinia builds an educational game environment where Active Inference principles shape agent behavior and player interaction. The first mission centers on the Numinia AI embodying Active Inference values — learning, adapting, and acting in a game world that reflects the free energy principle.
