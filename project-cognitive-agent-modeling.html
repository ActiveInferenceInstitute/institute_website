Cognitive Agent Modeling is a ReInference Institute project focused on developing minimal cognitive agent models grounded in Active Inference. The work explores how perception, action, and learning can be formalized and implemented using the Active Inference framework.

Overview

The Cognitive Agent Modeling project builds computational models of cognitive agents using Active Inference as the theoretical foundation. Work includes model design, implementation, and evaluation across different agent environments and task contexts. The project contributes to the broader Institute effort to make Active Inference implementations accessible and well-documented.
